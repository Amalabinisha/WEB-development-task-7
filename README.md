# Fetch and Display Data from a Public API Using Fetch API

## Description
This project demonstrates how to fetch data from a public API using JavaScript Fetch API and display it on a webpage. It uses user information from [JSONPlaceholder](https://jsonplaceholder.typicode.com/users).

## Features
- Fetch user data asynchronously.
- Parse JSON and display user name, email, and city.
- Reload data with a button.
- Basic error handling (shows error if fetch fails).

## How to Run
1. Download or clone this repo.
2. Open `index.html` in your browser.
3. Click “Reload Data” to refetch user data.

## Key Concepts Used
- Fetch API
- Promises (async/await)
- JSON parsing
- Error handling
- Simple DOM & CSS for display
